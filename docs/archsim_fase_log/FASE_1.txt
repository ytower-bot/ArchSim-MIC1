OBJETIVO
--------
Implementar 5 funções utilitárias de conversão entre bits e inteiros para
suportar operações básicas da arquitetura MIC-1.

FUNÇÕES IMPLEMENTADAS
---------------------

1. bits_to_int(int bits[], int size)
   - Converte array de bits (tamanho variável) para inteiro
   - Endianess: bit[0] = MSB
   - Validação: ponteiro e size
   
2. int_to_bits(int value, int bits[], int size)
   - Converte inteiro para array de bits (tamanho variável)
   - Endianess: bit[0] = MSB
   - Validação: ponteiro e size
   
3. address_to_int(int address[12])
   - Converte endereço de 12 bits para inteiro (0-4095)
   - Usa bits_to_int() internamente
   - Validação: ponteiro
   
4. int_to_address(int addr, int address[12])
   - Converte inteiro para endereço de 12 bits
   - Usa int_to_bits() internamente
   - Validação: ponteiro
   
5. copy_data(int dest[16], int src[16])
   - Copia array de 16 bits (src → dest)
   - Loop manual para compatibilidade
   - Validação: ambos ponteiros
