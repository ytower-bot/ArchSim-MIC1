FASE 1 - UTILITÁRIOS DE CONVERSÃO
Data: 27/11/2025
Status: Completa

OBJETIVO
Implementar funções utilitárias para conversões bit/inteiro usadas em toda arquitetura.

FUNÇÕES IMPLEMENTADAS
- bits_to_int(bits[], size) - Converte array de bits para inteiro
- int_to_bits(value, bits[], size) - Converte inteiro para array de bits
- address_to_int(address[12]) - Converte endereço de 12 bits para inteiro (0-4095)
- int_to_address(addr, address[12]) - Converte inteiro para endereço de 12 bits
- copy_data(dest[16], src[16]) - Copia arrays de dados de 16 bits

ARQUIVOS CRIADOS
- include/utils/conversions.h (declarações de funções)
- src/utils/conversions.c (implementações)
- src/tests/test_conversions.c (testes unitários)
